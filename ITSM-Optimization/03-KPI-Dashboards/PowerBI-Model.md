# Power BI ITSM Dashboard

## Data Sources
- Incidents Table (ServiceNow / Freshservice)
- Changes Table
- Asset CMDB

## KPIs Visualized
- MTTR by Priority
- SLA Compliance %
- Incident Volume by Category
- Problem Trends
- Change Success Rate

## Features
- Interactive slicers by Department, CI, Priority
- Alerts for SLA breach
- Trendline for Continual Service Improvement (CSI)
