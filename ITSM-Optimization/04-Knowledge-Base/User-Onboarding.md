## Objective
Standardize the onboarding process for all new municipal employees to ensure access, security, and compliance.

## Steps

1. **Account Creation**
   - Create Active Directory account
   - Assign Microsoft 365 license
   - Enable MFA and VPN access

2. **Hardware Provisioning**
   - Laptop/Desktop setup
   - Mobile device (if required)
   - Peripheral setup (printer, monitor, etc.)

3. **Application Access**
   - ERP / Finance system access
   - HR system access
   - Department-specific applications

4. **Security & Compliance**
   - Role-based access control
   - Enroll device in Intune (Endpoint Management)
   - Assign compliance policies

5. **Documentation**
   - Record in HR onboarding system
   - Update CMDB with device and user info

## Notes
- Automation scripts available for account provisioning
- SLA: Complete onboarding within 2 business days
